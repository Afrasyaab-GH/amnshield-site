AmnShield Docs

Welcome to the official documentation for AmnShield. This guide will help you configure the app for maximum digital protection and intentional focus.

1. Initial Setup & Permissions

To function effectively, AmnShield requires a few core Android permissions. Upon first launch, you will be guided to enable:

  • Accessibility Service: The backbone of AmnShield. It detects current apps and reads on-screen text to enforce keyword blocks.
  • Usage Access: Allows AmnShield to track screen time and enforce Launch Limits.
  • Display Over Other Apps: Required to show the block screen or warning overlay when an infraction occurs.

2. Core Features

App Blocker & Schedules

Navigate to the Blocks dashboard to manage your app restrictions. You can create grouped schedules (e.g., blocking all social media from 9 AM to 5 PM) or apply Launch Limits to restrict how many times you can open an app per day.

Focus Mode

Focus Mode is designed for deep work. When activated, it locks down all distracting apps for a set duration. You can configure which apps are considered "distracting" in the Focus Mode settings.

Keyword Blocker

Add specific words to your Keyword Blocklist. If these words appear on your screen (in a browser, social app, etc.), AmnShield will immediately block the view or navigate back.

3. Website Blocker & Filter Packs

AmnShield features a local Website Blocker that filters adult domains and distracting websites offline.

  • Enable Website Blocker: Go to the Blocks dashboard -> Website Blocker -> Toggle ON.
  • Filter Packs: Enable adult domain list (50k+ domains), social media domains, or gaming domains.
  • Exceptions: Add custom domains to the blocklist or whitelist to customize your filtering.

4. Error Reporting

If you encounter issues, AmnShield provides a local-only diagnostic tool. Go to Settings -> Error Reporting. You can view logs locally, and if you need help, you can export them and email them directly to our support team. No logs are ever sent automatically.
